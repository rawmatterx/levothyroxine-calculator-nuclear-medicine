# Levothyroxine Dosing & TSH-Target Calculator
### For Nuclear Medicine Physicians

A clinical decision-support web application built using Streamlit to assist Nuclear Medicine physicians in determining:

- Risk-adapted TSH targets  
- Suggested levothyroxine dosing ranges  
- Adjustment guidance and safety flags  

### Clinical Scope
Supports:
- Post-radioiodine hypothyroidism after treatment of hyperthyroidism
- Differentiated thyroid cancer (DTC) after thyroidectomy ± RAI

### ⚠️ Important
This tool is for use by qualified healthcare professionals only.  
It does not replace endocrinology consultation or institutional protocols.
